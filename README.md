# QA Manual Testing Project

## Project Overview
This project demonstrates manual testing of the educational platform PDDTest.kz.
The goal was to evaluate functionality, usability, validation, and localization quality.

## Scope of Testing
- Functional testing
- Validation testing
- Localization testing (RU / KZ)
- Dead link testing

## Tools Used
- Manual testing
- Dead Link Checker
- Chrome, Edge

## Key Outcomes
- Created structured test cases
- Reported bugs with severity and priority
- Identified critical validation and localization issues
